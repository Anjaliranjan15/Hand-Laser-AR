# ✋ Hand Laser AR 

Hand Laser AR is an interactive web-based Augmented Reality (AR) project that leverages real-time hand tracking to create visually engaging laser beam effects between fingertips. Built using MediaPipe and JavaScript, the application detects multiple hands through a webcam and dynamically renders glowing laser connections, gesture recognition, and particle effects.

The system uses computer vision to track 21 key hand landmarks and maps fingertip positions to generate synchronized beams between both hands. It also includes gesture detection (such as pinch and open hand) and a heads-up display (HUD) showing real-time metrics like number of hands detected, FPS, and gesture state.

This project demonstrates the integration of AI-based hand tracking with creative front-end visualization, making it suitable for applications in AR interfaces, gaming, and interactive UI systems.

---

##  Key Features

* Real-time hand tracking using MediaPipe
* Laser beam generation between corresponding fingertips
* Gesture detection (Pinch / Open hand)
* HUD with live data (FPS, hands detected, gesture, spread)
* Particle spark effects for enhanced visuals
* Webcam-based interaction (no external hardware required)

---

## Technologies Used
